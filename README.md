# FastAPI CRUD Task Manager

This is a simple backend project built using FastAPI that performs CRUD operations for managing tasks.

---

## Features

- Add a task
- View all tasks
- Update a task
- Delete a task
- Interactive API documentation using Swagger UI

---

## Tech Stack

- Python
- FastAPI
- Uvicorn
